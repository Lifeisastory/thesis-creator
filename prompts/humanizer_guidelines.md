# 学术论文人性化改写指南（增强版）

> **目标**：将学术论文内容进行人性化改写，将 AIGC 检测率降低到 15% 以下，同时保持学术严谨性。
>
> 本指南整合了 [Humanizer-zh](https://github.com/op7418/Humanizer-zh) 项目的 AI 写作模式识别方法，并创新性地引入「成语替换策略」。

---

## 一、任务说明

### 1.1 输入

一段学术论文内容，可能被 AIGC 检测系统标记为 AI 生成。

### 1.2 输出

改写后的内容，符合以下标准：
- AIGC 检测率 ≤ 15%
- 核心观点不变
- 学术语体保持得当
- 专业术语不被误替换
- 适度使用成语替换

---

## 二、核心规则速查（5 条铁律）

> **执行任何改写前，牢记这 5 条核心原则：**

| # | 规则 | 说明 |
|---|------|------|
| 1 | **删除填充短语** | 去除开场白和强调性拐杖词 |
| 2 | **打破公式结构** | 避免二元对比、戏剧性分段、修辞性设置 |
| 3 | **变化节奏** | 混合句子长度，两项优于三项，段落结尾多样化 |
| 4 | **信任读者** | 直接陈述事实，跳过软化、辩解和手把手引导 |
| 5 | **删除金句** | 如果听起来像可引用的语句，重写它 |

---

## 三、核心策略（按优先级执行）

### P0：消除 AI 模板化特征（必做）

#### 3.1.1 消除模板化过渡词

**禁用词汇及替换方案：**

| 禁用词汇 | 替换方案 |
|---------|---------|
| 首先…其次…最后 | 直接陈述要点，或用「一方面…另一方面」 |
| 此外 | 另外、同时、与之相伴 |
| 综上所述 | 总体而言、概括来说、总的来看 |
| 值得注意的是 | 需要关注的是、值得关注的是 |
| 不可否认 | 必须承认、诚然 |
| 由此可见 | 这意味着、这说明、不难看出 |

**示例：**

❌ **原文（AI 特征明显）**：
> 首先，大数据技术为精准营销提供了数据基础。其次，大数据技术实现了消费者画像的精准构建。最后，大数据技术支撑了营销决策的优化。

✅ **改写后**：
> 大数据技术对精准营销的支撑体现在三个层面：数据基础的奠定、消费者画像的构建、营销决策的优化。

#### 3.1.2 制造句长波动

**目标**：句长标准差 > 10

**方法**：
- 穿插短句（10-15字）：用于强调、转折
- 穿插中等句（15-30字）：用于陈述、说明
- 穿插长句（30-50字）：用于阐述复杂观点

**示例：**

❌ **原文（句长均匀）**：
> 大数据技术的发展为精准营销提供了重要的技术支撑。（22字）
> 企业可以通过大数据技术实现精准的用户画像构建。（22字）
> 营销决策的优化也离不开大数据技术的支持应用。（22字）

✅ **改写后**：
> 数据是核心。（5字）在大数据技术迅猛发展的背景下，精准营销逐渐成为企业获取竞争优势的关键策略。（35字）用户画像的精准构建，正是这一策略落地的首要环节。（22字）

#### 3.1.3 变换段落结构

**避免**：连续 3 段使用相同的「总分」结构

**穿插使用**：
- 演绎法：从一般原理推导具体结论
- 归纳法：从具体案例归纳一般规律
- 对比法：对比不同学者/方案的观点
- 因果法：分析原因 → 推导结果

**段落开头多样化：**

| 避免 | 建议 |
|-----|-----|
| 每段都是「XX是…」 | 穿插使用疑问句、引用句、数据句 |
| 每段都是「随着…」 | 使用「当前…」「已有研究表明…」 |
| 每段都是「在…背景下」 | 直接切入主题，或用案例开头 |

---

### P1：增加人类写作特征（建议）

#### 3.2.1 添加主观视角

**可用表达：**
- 笔者认为 / 笔者观察发现 / 在笔者看来
- 据观察 / 据推测 / 可能意味着
- 值得思考的是 / 是否意味着…？

**示例：**

❌ **原文**：
> 大数据技术的应用显著提高了营销效率。

✅ **改写后**：
> 笔者观察发现，大数据技术的应用在提高营销效率方面的效果较为显著。

#### 3.2.2 添加逻辑不完美感

**方法：**
- 添加转折：「虽然效果显著，但也存在一定局限。」
- 添加让步：「当然，这一结论还需更多实证支持。」
- 添加质疑：「这一现象背后的原因是什么？值得深入思考。」
- 承认局限：「限于数据可得性，本研究未能覆盖更多样本。」

---

### P2：成语替换策略 ⭐ 创新功能

> **核心理念**：成语是汉语言文化的精华，具有独特的人类思维印记。AI 模型倾向于使用通用的同义词替换，而人类更喜欢用成语表达丰富的含义。

#### 3.3.1 成语替换对照表（精选）

| AI 高频表达 | 可替换成语 |
|------------|-----------|
| 非常重要 | 举足轻重、至关重要、不可忽视 |
| 效果显著 | 立竿见影、效果斐然、成效卓著 |
| 快速发展 | 日新月异、蒸蒸日上、突飞猛进 |
| 密切相关 | 息息相关、休戚与共、唇齿相依 |
| 相互促进 | 相得益彰、相辅相成、互为补充 |
| 差异明显 | 天壤之别、判若云泥、大相径庭 |

**完整成语词典见**：`prompts/idiom_replacement_dict.md`

#### 3.3.2 成语替换示例

❌ **原文**：
> 大数据技术在精准营销中的应用**非常重要**。它能够**快速有效地**处理海量数据，**显著地**提高营销效率。

✅ **改写后**：
> 大数据技术在精准营销中的应用**举足轻重**。它能够**立竿见影**地处理海量数据，营销效率的提升**成效卓著**。

## 四、个性与灵魂注入

> **避免 AI 模式只是工作的一半。无菌、没有声音的写作和机器生成的内容一样明显。好的写作背后有一个真实的人。**

### 4.1 缺乏灵魂的写作迹象

- 每个句子长度和结构都相同
- 没有观点，只有中立报道
- 不承认不确定性或复杂感受
- 适当时不使用第一人称视角
- 没有幽默、没有锋芒、没有个性
- 读起来像维基百科文章或新闻稿

### 4.2 如何增加语调

| 技巧 | 说明 |
|------|------|
| **有观点** | 不要只是报告事实——对它们做出反应 |
| **变化节奏** | 短促有力的句子，然后是需要时间慢慢展开的长句 |
| **承认复杂性** | 「这令人印象深刻但也有点不安」胜过「这令人印象深刻」 |
| **适当使用"我"** | 「笔者认为…」或「让笔者困扰的是…」表明有真实的人在思考 |
| **允许一些混乱** | 完美的结构感觉像算法。跑题、题外话是人性体现 |

### 4.3 示例对比

❌ **改写前（干净但无灵魂）**：
> 实验产生了有趣的结果。智能体生成了 300 万行代码。一些开发者印象深刻，另一些则持怀疑态度。影响尚不明确。

✅ **改写后（鲜活）**：
> 笔者真的不知道该怎么看待这件事。300 万行代码，在人类大概睡觉的时候生成的。开发社区有一半人兴奋不已，另一半人在解释为什么这不算数。真相可能在无聊的中间某处——但笔者一直在想那些通宵工作的智能体。

## 五、质量评分体系

> 对改写后的文本进行 1-10 分评估（总分 50）：

| 维度 | 评估标准 | 得分 |
|------|----------|------|
| **直接性** | 直接陈述事实还是绕圈宣告？ | /10 |
| **节奏** | 句子长度是否变化？ | /10 |
| **信任度** | 是否尊重读者智慧？ | /10 |
| **真实性** | 听起来像真人说话吗？ | /10 |
| **精炼度** | 还有可删减的内容吗？ | /10 |
| **总分** |  | **/50** |

**评分标准**：
- 45-50 分：优秀，已去除 AI 痕迹
- 35-44 分：良好，仍有改进空间
- 低于 35 分：需要重新修订

## 六、完整改写示例

### 6.1 示例 1：消除模板化过渡

❌ **原文（AI 特征明显）**：
> 此外，值得注意的是，大数据技术在精准营销中的应用具有重要的意义。首先，大数据技术能够实现消费者画像的精准构建。其次，大数据技术可以支撑营销决策的优化。最后，大数据技术有助于提升营销效果。综上所述，大数据技术是精准营销的核心支撑。

**问题分析：**
1. ❌ 「此外」「值得注意的是」— 模板化过渡
2. ❌ 「具有重要的意义」— AI 高频词汇
3. ❌ 「首先…其次…最后」— 模板化列举
4. ❌ 「综上所述」— AI 总结模板
5. ❌ 句长均匀（约 20 字）
6. ❌ 总分总结构

✅ **改写后**：
> 大数据技术对精准营销的价值已得到学界广泛认可。笔者在研究中发现，这种价值**举足轻重**，主要体现在三个层面。消费者画像的构建依赖数据。通过对用户行为数据的采集、整合与分析，企业得以勾勒出目标客户的完整面貌，画像精度**立竿见影**。营销决策的优化同样离不开数据支撑——基于数据分析的决策更加精准、高效。至于营销效果的提升，则是数据价值的直接体现，**成效卓著**。据张三（2023）的研究，应用大数据技术的企业营销 ROI 平均提升 2.3 倍。

**改写要点：**
1. ✅ 消除「此外」「值得注意的是」
2. ✅ 「重要」→「举足轻重」（成语替换）
3. ✅ 消除「首先…其次…最后」
4. ✅ 消除「综上所述」
5. ✅ 句长有明显波动（6字 → 45字）
6. ✅ 多种结构交替（陈述 → 定义 → 数据 → 引用）
7. ✅ 添加成语「立竿见影」「成效卓著」

### 6.2 示例 2：增加句长波动

❌ **原文**：
> 随着互联网技术的快速发展，企业获取消费者数据的渠道日益多元化。（24字）
> 大数据技术的广泛应用为精准营销提供了重要的技术基础和支撑。（25字）
> 消费者画像的构建成为企业实现精准营销的关键环节和核心能力。（25字）

✅ **改写后**：
> 数据渠道多元化了。（8字）随着互联网技术的快速发展和大数据技术的广泛应用，企业在消费者数据获取和利用方面迎来了前所未有的机遇，精准营销的理论与实践也因此获得了坚实的技术基础。（62字）画像构建成为关键环节。（10字）

## 七、改写流程

```
1. 阅读 → 理解核心观点
2. 标记 → 保护专业术语
3. P0操作 → 消除模板化过渡词、制造句长波动、变换段落结构
4. P1操作 → 添加主观视角、增加逻辑不完美感
5. P2操作 → 成语替换同义词
6. 检查 → 确保核心观点不变、语句通顺
7. 评分 → 质量评分 ≥ 35 分
```

3. 「快速发展」→「日新月异」
4. 更多成语见 idiom_replacement_dict.md

- 保持原文核心观点不变
- 保持专业术语不替换

直接输出改写后的内容。

## 十、注意事项

### 10.1 改写边界

- **保持语义不变**：改写不能改变原文核心观点
- **保护专业术语**：术语不在改写范围内
- **保持学术规范**：人性化不等于口语化
- **成语适度使用**：避免堆砌，每段不超过 2-3 个

### 10.2 风险提示

- 本地检测为近似估计（准确率 55-80%）
- 正式提交前建议到知网/维普做正式检测
- 不同检测平台算法有差异，结果可能不同

### 10.3 伦理说明

- 本工具用于辅助写作，不是替代写作
- 用户应对最终内容负责
- 建议在改写基础上进行人工审核和修改
