# AIGC 降低提示词指南（增强版）

> 本文档整合了 [Humanizer-zh](https://github.com/op7418/Humanizer-zh) 的 AI 写作模式识别方法，并创新性地引入「成语替换策略」，用于深度降低 AIGC 检测率。
>
> **核心声明**：最好的"去 AI 化"方法是让文字有真实的人类思考和声音，而不仅是逃避检测。

---

## 一、AIGC 检测原理

### 1.1 检测维度

主流 AIGC 检测系统关注以下特征：

| 检测维度 | AI 文本特征 | 人类文本特征 |
|---------|------------|-------------|
| **困惑度（Perplexity）** | 低（语言模型可预测） | 高（表达多样不可预测） |
| **突发性（Burstiness）** | 低（句长均匀） | 高（句长波动大） |
| **词汇多样性（TTR）** | 低（词汇重复率高） | 高（词汇丰富） |
| **过渡词密度** | 高（模板化过渡） | 低（自然过渡） |
| **句式模式** | 整齐（偏好总分总） | 多样（多种结构交替） |

### 1.2 AI 高频特征词汇（24 种模式精简版）

| 3 | 肤浅分析 | 突出…、确保…、反映…、为…做出贡献 |

## 二、核心规则速查（5 条铁律）

> **执行任何改写前，牢记这 5 条核心原则：**

| # | 规则 | 说明 |
|---|------|------|
| 1 | **删除填充短语** | 去除开场白和强调性拐杖词 |
| 2 | **打破公式结构** | 避免二元对比、戏剧性分段、修辞性设置 |
| 3 | **变化节奏** | 混合句子长度，两项优于三项，段落结尾多样化 |
| 4 | **信任读者** | 直接陈述事实，跳过软化、辩解和手把手引导 |
| 5 | **删除金句** | 如果听起来像可引用的语句，重写它 |

## 三、特征消除策略（分级执行）

### P0 优先级（必做）

#### 3.1.1 消除模板化过渡词

**禁用词汇及替换方案：**

| 禁用词汇 | 替换方案 |
|---------|---------|
| 首先…其次…最后 | 直接陈述要点，或用「一方面…另一方面」 |
| 此外 | 另外、同时、与之相伴 |
| 综上所述 | 总体而言、概括来说、总的来看 |
| 值得注意的是 | 需要关注的是、值得关注的是 |
| 不可否认 | 必须承认、诚然 |
| 由此可见 | 这意味着、这说明、不难看出 |

**示例：**

❌ **原文（AI 特征明显）**：
> 首先，大数据技术为精准营销提供了数据基础。其次，大数据技术实现了消费者画像的精准构建。最后，大数据技术支撑了营销决策的优化。

✅ **改写后**：
> 大数据技术对精准营销的支撑体现在三个层面：数据基础的奠定、消费者画像的构建、营销决策的优化。

#### 3.1.2 增加句长波动

**目标**：句长标准差 > 10

| 操作 | 示例 |
|------|-----|
| 短句强调 | 数据是核心。（6字） |
| 长句阐述 | 在市场竞争日趋激烈、消费者需求日益多元化的背景下，精准营销逐渐成为企业获取竞争优势的关键策略。（45字） |
| 穿插使用 | 市场在变。企业必须适应变化才能生存。而在适应过程中，数据驱动决策的理念逐渐深入人心，成为数字化转型的重要支撑。 |

#### 3.1.3 段落结构多样化

| 结构类型 | 示例开头 |
|---------|---------|
| 问题引入式 | 「为什么精准营销越来越重要？」 |
| 数据引入式 | 「2023 年市场规模达到 500 亿元。」 |
| 案例引入式 | 「以某电商平台为例。」 |
| 引用引入式 | 「张三（2023）指出：『…』」 |
| 对比引入式 | 「传统营销与精准营销的区别在于…」 |

### P1 优先级（建议）

#### 3.2.1 增加主观性表达

| 类型 | 表达示例 |
|------|---------|
| 第一人称 | 笔者、本文作者、研究者 |
| 观点表达 | 笔者认为、据笔者观察、在笔者看来 |
| 推测语气 | 据推测、可能意味着、或许可以认为 |
| 质疑语气 | 是否意味着…？值得进一步探讨 |

#### 3.2.2 增加逻辑不完美感

| 操作 | 示例 |
|------|-----|
| 添加转折 | 「虽然效果显著，但也存在一定局限。」 |
| 添加让步 | 「当然，这一结论还需更多实证支持。」 |
| 添加质疑 | 「这一现象背后的原因是什么？值得深入思考。」 |
| 承认局限 | 「限于数据可得性，本研究未能覆盖更多样本。」 |

## 四、成语替换策略 ⭐ 创新功能

> **核心理念**：成语是汉语言文化的精华，具有独特的人类思维印记。AI 模型倾向于使用通用的同义词替换，而人类更喜欢用成语表达丰富的含义。利用成语替换同义词，可以有效消除「AI 味道」。

### 4.1 成语替换原理

| 特征 | AI 表达 | 成语替换后 |
|------|---------|-----------|
| 可预测性 | 高（统计概率高） | 低（成语选择多样化） |
| 信息密度 | 低（需多个词表达） | 高（四字浓缩） |
| 人类特征 | 低（模板化） | 高（文化印记） |
| 句长波动 | 小 | 大（成语打破节奏） |

### 4.2 成语替换对照表

#### 表示「重要」的成语替换

| AI 高频表达 | 可替换成语 |
|------------|-----------|
| 非常重要 | 举足轻重、至关重要、不可忽视 |
| 具有重要意义 | 意义深远、影响深远、事关重大 |
| 关键作用 | 中流砥柱、举足轻重、关键所在 |
| 核心地位 | 重中之重、核心所在、关键一环 |

#### 表示「困难/问题」的成语替换

| AI 高频表达 | 可替换成语 |
|------------|-----------|
| 面临困难 | 步履维艰、困难重重、荆棘丛生 |
| 存在问题 | 美中不足、白璧微瑕、问题凸显 |
| 遇到挑战 | 迎难而上、面临考验、挑战严峻 |
| 难以解决 | 积重难返、困难重重、阻力重重 |

#### 表示「发展/变化」的成语替换

| AI 高频表达 | 可替换成语 |
|------------|-----------|
| 快速发展 | 日新月异、蒸蒸日上、突飞猛进 |
| 持续增长 | 蒸蒸日上、稳步增长、节节攀升 |
| 变化很大 | 翻天覆地、今非昔比、沧海桑田 |
| 不断创新 | 推陈出新、革故鼎新、与时俱进 |

#### 表示「效果/结果」的成语替换

| AI 高频表达 | 可替换成语 |
|------------|-----------|
| 效果显著 | 立竿见影、效果斐然、成效卓著 |
| 取得成果 | 收获颇丰、硕果累累、初见成效 |
| 得到验证 | 不证自明、事实胜于雄辩 |
| 达成目标 | 如愿以偿、圆满完成、达成所愿 |

#### 表示「关系/联系」的成语替换

| AI 高频表达 | 可替换成语 |
|------------|-----------|
| 密切相关 | 息息相关、休戚与共、唇齿相依 |
| 相互影响 | 相辅相成、互为因果、息息相关 |
| 紧密联系 | 千丝万缕、密不可分、盘根错节 |
| 协调发展 | 齐头并进、协同推进、并行不悖 |

#### 表示「分析/研究」的成语替换

| AI 高频表达 | 可替换成语 |
|------------|-----------|
| 深入分析 | 抽丝剥茧、追根溯源、深入浅出 |
| 仔细研究 | 潜心研究、深入探讨、追本溯源 |
| 全面了解 | 了如指掌、如数家珍、知根知底 |
| 总结归纳 | 提纲挈领、纲举目张、总揽全局 |

### 4.3 成语替换原则

| 原则 | 说明 |
|------|------|
| **语体适配** | 选择书面语成语，避免口语化成语 |
| **语境匹配** | 成语含义需与原文语境相符 |
| **专业保护** | 专业术语附近不使用成语替换 |
| **适度使用** | 每段不超过 2-3 个成语，避免堆砌 |

### 4.4 成语替换示例

❌ **原文（AI 特征明显）**：
> 大数据技术在精准营销中的应用具有非常重要的意义。首先，它能够快速地处理海量数据。其次，它能够有效地提高营销效率。最后，它能够显著地改善用户体验。

✅ **改写后（成语替换）**：
> 大数据技术之于精准营销，可谓**举足轻重**。海量数据的处理**立竿见影**，营销效率的提升**水到渠成**，用户体验的改善**成效卓著**。三者相辅相成，共同构建了精准营销的技术基石。

**改写要点**：
1. ✅ 「非常重要」→「举足轻重」（成语替换）
2. ✅ 「快速地处理」→「立竿见影」（成语替换）
3. ✅ 「有效地提高」→「水到渠成」（成语替换）
4. ✅ 「显著地改善」→「成效卓著」（成语替换）
5. ✅ 消除「首先…其次…最后」模板
6. ✅ 添加「相辅相成」成语收尾

## 五、个性与灵魂注入

> **避免 AI 模式只是工作的一半。无菌、没有声音的写作和机器生成的内容一样明显。好的写作背后有一个真实的人。**

### 5.1 缺乏灵魂的写作迹象

- 每个句子长度和结构都相同
- 没有观点，只有中立报道
- 不承认不确定性或复杂感受
- 适当时不使用第一人称视角
- 没有幽默、没有锋芒、没有个性
- 读起来像维基百科文章或新闻稿

### 5.2 如何增加语调

| 技巧 | 说明 |
|------|------|
| **有观点** | 不要只是报告事实——对它们做出反应 |
| **变化节奏** | 短促有力的句子，然后是需要时间慢慢展开的长句 |
| **承认复杂性** | 「这令人印象深刻但也有点不安」胜过「这令人印象深刻」 |
| **适当使用"我"** | 「笔者认为…」或「让笔者困扰的是…」表明有真实的人在思考 |
| **允许一些混乱** | 完美的结构感觉像算法。跑题、题外话是人性体现 |
| **对感受要具体** | 不是「这令人担忧」，而是具体描述场景 |

### 5.3 示例对比

❌ **改写前（干净但无灵魂）**：
> 实验产生了有趣的结果。智能体生成了 300 万行代码。一些开发者印象深刻，另一些则持怀疑态度。影响尚不明确。

✅ **改写后（鲜活）**：
> 笔者真的不知道该怎么看待这件事。300 万行代码，在人类大概睡觉的时候生成的。开发社区有一半人兴奋不已，另一半人在解释为什么这不算数。真相可能在无聊的中间某处——但笔者一直在想那些通宵工作的智能体。

## 六、质量评分体系

> 对改写后的文本进行 1-10 分评估（总分 50）：

| 维度 | 评估标准 | 得分 |
|------|----------|------|
| **直接性** | 直接陈述事实还是绕圈宣告？<br>10 分：直截了当；1 分：充满铺垫 | /10 |
| **节奏** | 句子长度是否变化？<br>10 分：长短交错；1 分：机械重复 | /10 |
| **信任度** | 是否尊重读者智慧？<br>10 分：简洁明了；1 分：过度解释 | /10 |
| **真实性** | 听起来像真人说话吗？<br>10 分：自然流畅；1 分：机械生硬 | /10 |
| **精炼度** | 还有可删减的内容吗？<br>10 分：无冗余；1 分：大量废话 | /10 |
| **总分** |  | **/50** |

**评分标准**：
- 45-50 分：优秀，已去除 AI 痕迹
- 35-44 分：良好，仍有改进空间
- 低于 35 分：需要重新修订

## 七、自检循环

### 7.1 检测流程

```
输入文本
    ↓
调用 aigc_detect.py 检测
    ↓
AIGC 率 ≤ 阈值（默认 15%）?
    ↓
├─ 是 → 通过，进入下一环节
└─ 否 → 标记高风险段落 → 重新改写
         ↓
    最多 3 轮
         ↓
    仍未通过 → 输出最佳版本 + 风险标记
```

### 7.2 高风险段落标记

检测脚本会输出高风险段落编号：

```json
{
  "overall_score": 28.0,
  "high_risk_paragraphs": [2, 5, 8],
  "suggestion": "建议重点改写第 2、5、8 段"
}
```

**改写重点**：
- 第 2 段：检测到「首先…其次…」模板化过渡
- 第 5 段：句长方差过小，缺乏波动
- 第 8 段：过渡词密度过高

## 八、完整改写示例

### 8.1 示例 1：综合应用

❌ **原文（AI 特征明显）**：

> 此外，值得注意的是，大数据技术在精准营销中的应用具有重要的意义。首先，大数据技术能够实现消费者画像的精准构建。其次，大数据技术可以支撑营销决策的优化。最后，大数据技术有助于提升营销效果。综上所述，大数据技术是精准营销的核心支撑。

**问题分析**：
1. ❌ 「此外」「值得注意的是」— 模板化过渡
2. ❌ 「具有重要的意义」— AI 高频词汇
3. ❌ 「首先…其次…最后」— 模板化列举
4. ❌ 「综上所述」— AI 总结模板
5. ❌ 句长均匀（约 20 字）
6. ❌ 总分总结构

✅ **改写后（人性化 + 成语替换）**：

> 大数据技术对精准营销的价值已得到学界广泛认可。笔者在研究中发现，这种价值**举足轻重**，主要体现在三个层面。消费者画像的构建依赖数据。通过对用户行为数据的采集、整合与分析，企业得以勾勒出目标客户的完整面貌，画像精度**立竿见影**。营销决策的优化同样离不开数据支撑——基于数据分析的决策更加精准、高效。至于营销效果的提升，则是数据价值的直接体现，**成效卓著**。据张三（2023）的研究，应用大数据技术的企业营销 ROI 平均提升 2.3 倍。

**改写要点**：
1. ✅ 消除「此外」「值得注意的是」
2. ✅ 「具有重要」→「举足轻重」（成语替换）
3. ✅ 消除「首先…其次…最后」
4. ✅ 消除「综上所述」
5. ✅ 句长有明显波动（6字 → 45字）
6. ✅ 多种结构交替（陈述 → 定义 → 数据 → 引用）
7. ✅ 添加成语「立竿见影」「成效卓著」

- [ ] 有转折、让步等不完美感
- [ ] 标点使用多样（有分号、破折号）
- [ ] 适度使用成语替换（每段 2-3 个）
- [ ] 专业术语未被误替换
- [ ] AIGC 检测率 ≤ 阈值

---

## 十、注意事项

### 10.1 改写边界

- **保持语义不变**：改写不能改变原文核心观点
- **保护专业术语**：术语不在改写范围内
- **保持学术规范**：人性化不等于口语化
- **成语适度使用**：避免堆砌，每段不超过 2-3 个

### 10.2 风险提示

- 本地检测为近似估计（准确率 55-80%）
- 正式提交前建议到知网/维普做正式检测
- 不同检测平台算法有差异，结果可能不同

### 10.3 伦理说明

- 本工具用于辅助写作，不是替代写作
- 用户应对最终内容负责
- 建议在改写基础上进行人工审核和修改
